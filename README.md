# Consumer-Retail-Behavior-Analysis

## Project Overview
* This project analyzes consumer retail behavior using advanced statistical and machine learning techniques to identify key factors affecting sales, demographics, and promotional effectiveness.

## Dataset
Uses a real-world retail transactions dataset with over 3,900 records.

Includes sales data, customer demographics, and promotional campaign details.

## Key Features
Regression models (linear and multiple) to estimate sales drivers.

Clustering to segment customer purchase behavior.

Logistic regression for discrete probability outcomes.

Use of machine learning models (e.g., Random Forest) to improve prediction accuracy (RÂ²=0.84).

## Dashboards & Visualizations
Interactive Tableau dashboards presenting sales trends and promotional impacts.

Visual explorations of demographic influences on buying patterns.

## Insights
Identification of key promotional activities that drive sales.

Segmentation of customers for targeted marketing.

Analysis of product attributes linked to higher sales performance.
